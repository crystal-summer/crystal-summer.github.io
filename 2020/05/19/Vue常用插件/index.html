数字滚动效果插件(vue-countTo)

vue-countTo是一个无依赖、轻量级的组件，可以设置 startVal 和 endVal，它会自动判断计数或倒计时。

富文本编辑器(wangEditor)


wangEditor是基于 javascript 和 css 开发的 Web 富文本编辑器， 轻量、简洁、易用、开源免费。

踩坑一：复制文档里面的内容，粘贴到富文本编辑器，传到后台的文本带有前端样式。解决方法：去除word粘贴的样式 ，问题是可能去除掉一部分内容。

踩坑二：上传视频功能不全。解决方法：替换掉wangEditor.min.js文件，在wangEditor.js里面添加上传视频的参数。
