nvm

posted 2019-02-23 in Other ， nvm count： 996 readingTime ≈ 1 分钟

介绍

  nvm 是 node 管理工具，当我们在日常开发中遇到这种问题：手上有好几个项目，每个项目的需求不同，进而不同项目必须依赖不同版的 NodeJS 运行环境的时候，nvm 就解决了我们的困扰。
  注意：在安装 nvm 之前必须先卸载到电脑里的 node。
  Windows 安装：nvm下载地址

  • nvm-noinstall.zip：绿色免安装版，但使用时需进行配置
  • nvm-setup.zip：安装版，推荐使用

安装

  正常安装即可，其中需要注意的地方是第一次选择安装文件夹的时候，这里是存放安装所必需的的文件文件（如D:\nvm）；第二次选择文件夹的地方，这里需要自己新建一个文件夹，比如D:\nvmSys，选择这个文件并且安装完成后，D盘将找不到这个文件，但是这个文件路径将会写入环境变量。

淘宝镜像

  nvm安装完成后，打开nvm文件下settings文件，添加下载镜像指向淘宝，可以加快我们之后npm的速度。
