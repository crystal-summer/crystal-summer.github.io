进度条(nprogress)

nprogress是一个youtube、Medium等网站上的小进度条插件。
1. 实现原理：
  一个页面或者接口的进度计算很复杂，即使计算出来也需要消耗非常大的性能，而 nprogress 是一个虚假的进度条，一开始进入处理的时候会显示出loading的效果，等捕获到方法结束，就会释放，进度完成100%，为了防止过渡过程生硬，到达100%之前会有一个缓慢加载的过程，虚假的进度条并不会消耗太多的性能。
  nprogress 核心原理就是每隔trickleSpeed毫秒调用NProgress.inc()方法，该方法是增加进度条的百分比，如果百分比是0<=status<0.2，就在原有基础加上0.1，如果百分比是0.2<=status<0.5，就在原有基础加上0.04，如果百分比是0.5<=status<0.8，就在原有基础加上0.02，如果百分比是0.8<=status<0.99，就在原有基础加上0.005，这里做了限制，百分比最多加到0.994，然后调用NProgress.done()方法直接将百分比设置成1，最后通过NProgress.set()方法将 status 百分比渲染到元素上，动态改变进度条 bar 的 translate 或者 margin-left。
2. 参数解析：

  • NProgress.configure()：进度条参数配置
  • NProgress.start()：进度条开始；
  • NProgress.done()：进度条结束消失，入参 true 表示强制关闭进度条；
  • NProgress.set()：设置进度条的百分比，数值范围 0.0 到 1.0；
  • NProgress.inc()：增加进度条进度，不传入参随机递增，传入参就会获取当前进度加上入参递增，但是永远不会达到100%；
  • NProgress.status：获取当前进度条的进度。

数字滚动(vue-count-to)

vue-countTo是一个无依赖、轻量级的组件，可以设置 startVal 和 endVal，它会自动判断计数或倒计时。
1. 实现原理：
  利用requestAnimationFrame或者setTimeOut实现初始值到结束值的递增或者递减，requestAnimationFrame当准备更新动画时应该调用此方法，这将使浏览器在下一次重绘之前调用传入给该方法的动画函数(即回调函数)，回调函数的入参是当前时间戳，如果浏览器不兼容该方法，可以使用setTimeOut来代替。
2. 参数解析：
