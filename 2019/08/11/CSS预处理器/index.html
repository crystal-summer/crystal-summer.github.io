CSS预处理器

posted 2019-08-11 in Web ， CSS count： 13k readingTime ≈ 11 分钟

前言

　　对于 CSS 样式，公司要求统一，主推 Sass ，其次是 Less ，而 Stylus 仅仅是个人感兴趣。

Sass

　　Sass 是一款强化 CSS 的辅助工具，它在 CSS 语法的基础上增加了变量、嵌套、混合、导入等高级功能，这些拓展令 CSS 更加强大与优雅。使用 Sass 以及 Sass 的样式库（如 Compass）有助于更好地组织管理样式文件，以及更高效地开发项目。

  1. 特色
    • 完全兼容 CSS3；
    • 在 CSS 基础上增加变量、嵌套 (nesting)、混合 (mixins) 等功能；
    • 通过函数进行颜色值与属性值的运算；
    • 提供控制指令 (control directives)等高级功能；
    • 自定义输出格式。
  2. 语法格式
    • .scss：这种格式仅在 CSS3 语法的基础上进行拓展，推荐使用这种格式；
    • .sass：缩进格式，它使用 “缩进” 代替 “花括号” 表示属性属于某个选择器，用 “换行” 代替 “分号” 分隔属性。

变量

　　可以把反复使用的 css 属性值定义成变量，然后通过变量名来引用它们，而无需重复书写这一属性值。
　　变量名既可以用中划线也可以用下划线，看个人喜好。
　　变量的作用域只能在当前的层级上有效果。
　　可以使用 !global 关键词来设置全局变量。
　　可以使用 !default 来规定变量名的默认值，如果变量名无值的话就显示默认值，有值就显示那个值即可。

嵌套

　　嵌套可以避免重复书写父选择器。
　　父选择器的标识符是 & 。
　　> 子层紧跟的元素。
　　+ 同层紧跟的元素。
　　~ 同层所有的元素。

导入@import

　　在 CSS中 使用 @import 导入其他的 CSS 文件，浏览器需要发起额外的下载请求去下载被导入的 CSS 文件，导致页面加载很慢，但是 Sass 导入无需发起额外的请求，页面加载会快一点。

注释

　　静默注释：//，编译后不会出现在 CSS 文件中；
　　标准注释：/*......*/，编译后会出现在 CSS 文件中，但是如果出现在原生 CSS 不允许出现的地方，该注释就会被抹去，在压缩模式下标准注释也不会显示，除非使用多行缓冲注释，即加上感叹号 /*! */；
　　文档注释：/// 也是属于静默注释，直接写在文档的正上方。

混合器

　　混合器可实现大段样式的重用，混合器使用 @mixin 标识符定义，通过 @include 来使用。
　　可以通过 @include 引入混合器时给混合器传参，来定制混合器生成的精确样式。当 @include 混合器时，参数其实就是可以赋值给 css 属性值的变量；为了在 @include 混合器时不必传入所有的参数，我们也可以给参数指定一个默认值。

注意：不可过度使用混合器，大量的重用可能会导致生成的样式表过大，页面加载缓慢。

选择器继承

　　选择器继承是说一个选择器可以继承为另一个选择器定义的所有样式，这和混合器一样，也是减少样式重复的主要特性。选择器继承通过 @extend 语法实现。比如当一个 div 元素定义了一个类，名为 A，还有一个类，名为 B，A 中 @extend .B，就意味着这个 div 有两个类名，A 和 B。
　　当前选择器不仅会继承另一选择器自身的所有样式，还会把任何和另一选择器有关的组合选择器样式也继承下来。

插值语句

　　通过 #{} 插值语句可以在选择器或属性名中使用变量。

占位符

　　占位符类是一种特殊类型的类，它只在被继承使用时才会被编译到 CSS，这样可以保持编译后的 CSS 整洁。

控制指令

  1. @if @else if @else：
  2. @each
  3. @for
    　　两种格式：@for $var from <start> through <end>，或者 @for $var from <start> to <end>，区别在于 through 与 to 的含义：
  • through：条件范围包含 <start> 与 <end> 的值；
  • to：条件范围只包含 <start> 的值不包含 <end> 的值。
  1. @while

Less

  1. 变量：用法同 Sass，写法不同，Less 使用 @，Sass 使用 $；
  2. 嵌套：同 Sass；
  3. 导入：同 Sass；
  4. 注释：同 Sass；
  5. 混合器：原理同 Sass，但是写法不一样，Less 是将类名像函数一样调用即可，Sass 是使用 @mixin @include

Stylus

　　Stylus 采用缩进格式，它使用 “缩进” 代替 “花括号” 表示属性属于某个选择器，用 “换行” 代替 “分号” 分隔属性，和 Sass 的 .sass 语法一样。

  1. 变量：用法同 Sass，写法不同，Less 可以 font-size = 14px，也可以 $font-size = 14px，Sass 使用 $；
  2. 嵌套：用法同 Sass；
  3. 导入：用法同 Sass；
  4. 注释：用法同 Sass；
  5. 混合器：用法同 Less；
  6. 选择器继承：基本同Sass，但是Sass不能继承嵌套的选择器；
  7. 插值：Stylus 使用 {}，而 Sass 使用 #{}；
  8. 控制指令：
    • 条件：Stylus 使用 if else if else，而 Sass 使用 @if @else if @else；
    • 迭代：Stylus 使用 for <val-name> [, <key-name>] in <expression>，而 Sass 使用 @for $var from <start> through <end> 或者 @for $var from <start> to <end>；
